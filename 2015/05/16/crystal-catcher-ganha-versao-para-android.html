    Crystal Catcher ganha versão para Android

    Tweet

    Um runner com uma pitada de puzzle! Essa é a melhor maneira para descrever Crystal Catcher, para iPhone e iPad.

    Foi assim que em 2012 descrevemos o game Crystal Catcher da desenvolvedora brasileira Crazy Minds Game Studio, agora a equipe de desenvolvimento entrou em contato conosco para nos contar uma boa novidade: o game foi ançado para Android na Google Play.

    Para quem curte um endless runner, Crystal Catcher é uma ótima pedida.
